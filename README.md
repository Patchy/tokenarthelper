# Token Art Helper

A FoundryVTT module for creating portrait and token art. Mostly for my friend who dms my pf2e campaign who really wanted a tool like tokenizer to make tons of token art in advance but not tied to an actor, and with the ability to choose the filename it saves as.

## Features

- Open the editor from the scene controls toolbar
- Edit portrait and token images side-by-side with a full layer editor
- Type in your own filenames for the portrait and token images
